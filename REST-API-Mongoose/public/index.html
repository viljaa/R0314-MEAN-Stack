Airbnb listing data REST-API

API for fetching Airbnb listing data from MongoDB dataset

HTTP-methods:

  • GET one listing with id route: /api/getListing/<add id>
  • GET all listings route (shows 20 first results): /api/getListings
  • POST new listing to DB route: /api/addListing/id=<add id>/name=<add name>/country=<add country>
  • DELETE listing from DB: /api/deleteListing/<add id>
  • PATCH - update existing listing: /api/updateListing/id=<add id>/name=<add name>/country=<add country>
